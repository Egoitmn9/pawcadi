# 📸 Put your Lick Mat photos here

The website looks for these exact file names. Drop your photos in this
folder using **these names** and they'll appear automatically — no code
changes needed. Until you do, a friendly placeholder is shown instead.

| File name        | Where it shows up on the site            | Suggested shot                          |
|------------------|------------------------------------------|-----------------------------------------|
| `hero.jpg`       | Big image at the top (hero)              | A dog happily licking the mat           |
| `product-1.jpg`  | Main product image (Shop section)        | Clean studio shot of the mat            |
| `product-2.jpg`  | Product thumbnail #2                     | Mat from another angle / in use         |
| `product-3.jpg`  | Product thumbnail #3                     | Close-up of the texture / nubs          |
| `product-4.jpg`  | Product thumbnail #4                     | Mat with food spread on it / in the tub |

## Tips
- **Format:** `.jpg` is expected. If you want to use `.png`/`.webp`, either rename
  your file to `.jpg`, or update the `src="..."` paths in `index.html`.
- **Shape:** product images look best **square** (e.g. 1000×1000 px). The hero
  image looks best **portrait / tall** (e.g. 1000×1250 px). They'll be cropped
  to fit automatically.
- **Size:** keep each file under ~500 KB so the page loads fast. You can shrink
  photos at https://squoosh.app (free, no signup).
